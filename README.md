# 039 Overloading

[![039 Overloading](https://img.youtube.com/vi/Gjw7KuWwaIM/0.jpg)](https://www.youtube.com/watch?v=Gjw7KuWwaIM)

the Person class
- should have a constructor that takes a `firstName`, `lastName` and `age` and sets them on the corresponding class-level fields
- should overload the constructor with one that only takes a `firstName` and `lastName` and sets them on the corresponding class-level fields
